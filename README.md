# Customer Churn Classification

## Overview
Predict whether a bank customer will leave (churn) or stay. Helps improve retention and target strategies.

## Dataset
Contains customer info: CreditScore, Geography, Gender, Age, Tenure, Balance, NumOfProducts, etc.

## Goal
Binary classification:
- 1 → Customer left  
- 0 → Customer stayed

## Approach
- Data cleaning & preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature encoding & scaling  
- Train ML models: Random Forest
- Evaluate with Accuracy, Precision, Recall, F1, ROC

## Usage
Load dataset → preprocess → train model → predict churn
